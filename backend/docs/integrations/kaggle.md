# Como obter sua API Key do Kaggle

1. Vá até o site [Kaggle.com](https://www.kaggle.com) e faça login.
2. Clique na sua foto de perfil no canto superior direito e vá em **Settings**.
3. Role até a seção **API**.
4. Clique em **Generate New Token**.
5. De um nome ao Token e clique em **Generate**.
6. Copie o **API TOKEN** e Cole no campo API KEY do formulário de Integração.
